Current Projects

Mountain Review

Mountain Review is a full-stack application where users can share their experiences at a particular ski resort. Anyone can sign up for an account to create a review and rate a particular place. They also have the ability to update or delete a review that they created. The back-end is built with Ruby on Rails, and the front-end with React.

View Mountain Review Here

The Library is a full-stack application containing user-added books. Anyone can search and view the books in the library as well as the author, genre, and a brief description of any book. If a user would like to contribute, they can create an account and add their own books to the library. They can also update or delete a book if there are corrections that need to be made. The back-end is built with MongoDB and Mongoose, and the front-end with React.
